What is a buildpack?

buildpack === installation script

  • Retrieves and install dependencies (NPM, etc).
  • Outputs generated assets or compiled code.
  • Configures the server or webserver.
  • Heroku has official buildpacks for: Ruby, Python, Java, Clojure, Node.js, Scala, Go and PHP
  • Heroku's buildpacks are open-source.

Dokku

Your own mini-Heroku on any cloud hosting provider

  • Open-source project released in 2013 by Jeff Lindsay.
  • 13,000+ stars and 260+ contributors on Github.
  • Auto-magically configures a Docker container with Ngnix.
  • Zero-downtime deployments.
  • Can configure multiple sub-domains.
  • Uses Heroku buildpacks, custom buildpacks or Dockerfiles.
  • Awesome documentation. Easy to understand.
  • Uses plugins to extend its capabilities...

Plugin Types:

  • Databases
  • Caching
  • Queuing
  • Process Managers
  • Monitors / Metrics
  • Security
  • Domain Admin
  • Container Utilities
  • File Utilities
  • Maintenance
  • and more...
