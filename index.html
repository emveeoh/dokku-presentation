What is Dokku & why should I care?

tl;dr => Heroku inside a Docker container. Fast, easy deployments.

Multiple apps on one domain. Not tied to any one cloud hosting provider. Can be deployed in many...
Caveats and what Dokku is *not* good for:
- must re-push apps after server reboot (Docker changed the way networking works... explain this...)
- only as secure as an open source project can be
- Best for: small deployments, internal apps, testing
What are buildpacks. Languages supported.
Plugins are available for quick installation and configuration of popular tools. Give current list of plugins. We will use the plugin Dokku-Letsencrypt to secure our website (HTTPS).
